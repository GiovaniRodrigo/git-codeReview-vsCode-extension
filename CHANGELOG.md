# Changelog

All notable changes to the **Code Review Design Extension** will be documented in this file.

## [1.0.0] - 2026-05-10

### Added
- **Architectural Analysis Engine**: Support for SRP, OCP, LSP, ISP, DIP and Clean Architecture rule validation.
- **Interactive Dashboard**: Executive overview of project quality and technical debt.
- **Git Integration**: Full support for reviewing Pull Requests, commits, and branches.
- **Review Session Management**: Create, manage, and track review sessions with audit logs.
- **Findings System**: Register technical findings with severity, status, and code linking.
- **Telemetry & Metrics**: Track quality scores, error frequency, and correction times.
- **Material Design 3 UI**: Modern and responsive interface for all panels.
- **Onboarding Tour**: Interactive guide for new users to explore extension features.
- **Persistence**: Local database with NDJSON audit trails and backup/sync capabilities.
