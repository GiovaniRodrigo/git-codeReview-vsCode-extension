# Changelog

## 0.0.1

- Added local Git TreeView for branches, tags, commits, changed files, and native VS Code diffs.
- Added review Webview with commit metadata, statistics, filters, search, and multi-select.
- Added GitHub remote detection, authentication, pull request listing, PR status, and checks.
- Added PR review actions: approve, request changes, general comment, and file/line review comments.
- Added GitHub-style patch Webview.
- Added branch comparison, reviewed commit history, unreviewed commit view, cache, background refresh, and multi-root active-folder detection.
- Added package, compile, lint, and test scripts.
