# Changelog v1.0.1 - Features & Refinements

## [1.0.1] - 2026-05-10

### Added
- **Multilingual Support (i18n)**: Full localization support with initial English (en) and Portuguese (pt-BR) translations.
- **Collaboration Menu**: New alignment for collaboration features between reviewers and developers.
- **Access Control**: Role-based access logic (Avaliador vs. Dev) for the onboarding process.
- **Smart Sidebar**: Resizable sidebars and visibility controls for a cleaner workspace.

### Changed
- **Onboarding Update**: Refined the guided tour for different user roles.
- **UI/UX Refinements**: Improved border rendering and menu alignments (Intelligence, Comments).

### Fixed
- Various UI bugs in the sidebar and dashboard.
- Logic for hidden bars and workspace integration.
