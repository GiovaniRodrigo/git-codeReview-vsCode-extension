# Pull Request Review

## Objetivo

Analisar impacto arquitetural e técnico das alterações.

---

# Informações da PR

- branch origem
- branch destino
- autor
- reviewer
- métricas
- score
- risco

---

# Recursos

## Diff Inteligente

Além de linhas alteradas:

- impacto arquitetural
- dependências
- SOLID
- Clean Architecture
- DDD

# Status

- OPEN
- IN_REVIEW
- NEEDS_CHANGES
- FIXED
- APPROVED
- REOPENED

# Telemetria

- violações recorrentes
- score por PR
- histórico
