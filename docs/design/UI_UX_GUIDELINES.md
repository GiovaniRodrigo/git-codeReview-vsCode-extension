# UI/UX Guidelines

## Objetivos

- Reduzir carga cognitiva
- Melhorar fluxo de revisão
- Fornecer feedback imediato
- Facilitar navegação
- Integrar review ao VS Code

---

# UI

## Componentes

- Cards
- Chips
- Painéis
- Snackbar
- Tabs
- Tooltips
- Badges
- Timeline

---

# UX

## Fluxo ideal

```text
Abrir PR
↓
Executar análise
↓
Visualizar problemas
↓
Comentar
↓
Corrigir
↓
Revalidar
↓
Aprovar
```

---

# DX (Developer Experience)

A ferramenta deve:

- evitar excesso de cliques
- evitar troca de contexto
- fornecer atalhos
- fornecer navegação rápida
- manter feedback contínuo
